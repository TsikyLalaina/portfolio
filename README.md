## Lalaina — Web Developer Portfolio

React + Vite + TypeScript portfolio with motion-first UI and a 3D hero scene.

### Tech
- React 19, Vite, TypeScript
- Framer Motion (animations), React Three Fiber + Drei (3D), postprocessing (Bloom/Vignette)
- Lenis (smooth scrolling), Zustand (state), Lottie (vector animations)

### Run
```bash
npm run dev
```

### Build
```bash
npm run build
npm run preview
```

### Deploy
- Vercel or Netlify — import repo, use default Vite settings.

### Customize
- Edit `src/components/Sections.tsx` to add real projects.
- Tweak 3D hero in `src/components/Hero3D.tsx`.
